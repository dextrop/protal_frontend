One Time Setup


1. Download ubuntu 18.0.4 ( Link for download )
2. Install ubuntu ( Install with window / Install without windows )
3. Download Pycharm ( required for python )
4. Download Phpstorm ( required for web developement )
5. Download script for initial setup.
6. Once the script is downloaded open terminal and run ~/path-to-script/.install.sh
